HTML & CSS Quick Reference Guide

This guide includes common HTML tags and essential CSS properties. Use it as a cheat sheet for your web projects.

HTML Tags

<h1> to <h6> - Headings

Defines headings of different levels.

<h1>Main Heading</h1>
<h2>Subheading</h2>

<p> - Paragraph

Used to define paragraphs.

<p>This is a paragraph.</p>

<a> - Anchor (Link)

Defines a hyperlink.

<a href="https://example.com">Visit Site</a>

<ul>, <ol>, <li> - Lists

Creates unordered or ordered lists with list items.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<img> - Image

Embeds an image in the page.

<img src="image.jpg" alt="Description">

<div> - Division

Container element used to group content.

<div class="container">
  <p>Some text here</p>
</div>

<span> - Inline Container

Used for styling inline portions of text.

<p>This is <span class="highlight">important</span> text.</p>

<br> - Line Break

Inserts a single line break.

Hello<br>World!

<pre> and <code> - Code Formatting

Used to display formatted code blocks.

<pre><code>Some code here</code></pre>

<link> - Stylesheet Link

Links external CSS files to HTML documents.

<link rel="stylesheet" href="styles.css">

CSS Properties

color

Changes the text color.

p {
  color: #333;
}

background-color

Sets the background color of an element.

body {
  background-color: #f0f0f0;
}

font-family

Sets the font type.

h1 {
  font-family: Arial, sans-serif;
}

font-size

Defines the size of the font.

p {
  font-size: 16px;
}

margin

Sets the space around elements.

h2 {
  margin-bottom: 20px;
}

padding

Controls the space inside elements.

.box {
  padding: 10px;
}

border

Defines a border around elements.

div {
  border: 1px solid #ccc;
}

text-align

Aligns text horizontally.

h1 {
  text-align: center;
}
